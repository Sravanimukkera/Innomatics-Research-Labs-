# Movie Ticket Booking System (FastAPI)

This project is a FastAPI backend application for booking movie tickets.

## Features
- Add, update, delete movies
- Search movies
- Sort movies by rating
- Pagination
- Book tickets
- Checkout system

## Tech Used
- Python
- FastAPI
- Pydantic

## How to Run
uvicorn main:app --reload
